벤더 프리픽스란?

벤더 프리픽스는 브라우저마다 따로 놀던 CSS3의 속성을 각각의 브라우저들이 인식할 수 있게 해주기 위해서 만들어졌다. CSS 표준안으로 확정되지 않았거나 특정 브라우저에서만 지원되는 CSS 속성을 사용하고싶을 때, 벤더 프리픽스를 사용하면 그 기능이 포함되어 있지 않은 브라우저에서도 사용할 수 있게 해준다.

벤더 프리픽스 사용방법

속성 앞에 브라우저의 접두어를 넣은 코드를 추가해주면 된다. 예를 들어, 사파리 브라우저에서 지원하는 border-radius 속성을 사용할 때는
-webkit-border-radius라는 벤더 프리픽스를 붙여서 사용한다.

HTML 요소에도 벤더 프리픽스가 사용될 수 있다. 예를 들어, iOS 기기에서 스크롤 효과를 제어하기 위해 -webkit-overflow-scrolling이라는 벤더 프리픽스가 사용될 수 있다.

벤더 프리픽스 장점

  • 각 브라우저마다 해당 속성을 구현하는 방식이 다르기 때문에, 벤더 프리픽스를 사용하면 각 브라우저에서 해당 기능이 올바르게 작동할 수 있도록 보장한다.

벤더 프리픽스 단점

  • 벤더 프리픽스의 이르이 다르기 떄문에 코드의 가독성과 유지 보수가 어렵다.
  • 최근에는 벤더 프리픽스의 사용을 지양하고, 대부분의 브라우저에서 호환되는 css속성과 html요소만 사용하는 것이 권장된다.
